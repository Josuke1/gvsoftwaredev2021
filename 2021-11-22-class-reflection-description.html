Class Reflection

Assignment Description:

The class was assigned 4 separate topics for the week of 10/25/21 which was led by JacobT and David.

The 4 topics included:

  • Class/goal reflections
  • Software development topics
  • Software development methodologies
  • David sharing his work experience

At the beginning of the year each student had to write down their class goals and this first topic is reflecting on those goals now that we are almost done with the semester, and it also included a reflection of the class as a whole. For example, what would you do differently if you were to lead the class, what do you think is going well, etc.

Each student was given a different type of software development such as data science or web development and had to find articles to read and write about based on the topic they were given for the second topic. They could write things down they knew already, didnâ€™t know, found interesting, etc.

For the third topic, each person was given a type of software development methodology which is basically the work method that different people/companies use such as Kanban and Scrum, or Agile and Waterfall. They were to reflect on this just like topic #2 and this is important because we are using some of those methodologies in this class

For the final topic, David shared his work experience with the class and showed his screen and what he typically does everyday as someone who actually works in the software development field. Students wrote down questions they had for him or things they found interesting and asked him those questions
